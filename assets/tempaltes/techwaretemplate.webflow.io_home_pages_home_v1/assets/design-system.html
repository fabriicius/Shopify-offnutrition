05 — Motion & Interaction

Motion with intent.

Every animation in the system is a reusable primitive. Pair these on any element by combining a keyframe class with an animate-on-scroll trigger.

Entry keyframes
fadeInUpBlur
animate-entry
.animate-entry.delay-100

Opacity 0→1, translateY 20px→0, blur 10px→0. 1s cubic-bezier(.2,.8,.2,1).

animationIn
animate-on-scroll
[animation:animationIn_0.8s_ease-out_0.1s_both]

Fires when the element enters the viewport. Paused until observed by IntersectionObserver.

text-clip-anim
.text-clip-anim > span
